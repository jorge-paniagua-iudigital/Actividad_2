## Descripción
En esta actividad se desarrolló un proceso de ingestión, almacenamiento y validación de datos utilizando Databricks Community Edition, Apache Spark y SQL.

El proyecto incluye:

- Diseño del esquema de datos
- Configuración del entorno Databricks
- Carga de datos desde Kaggle
- Creación de tabla en Spark SQL
- Validaciones utilizando PySpark y SQL
- Comparación entre SQL y Spark

## Dataset utilizado
Netflix Movies and TV Shows Dataset (Kaggle)
